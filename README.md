# E-LEARNING-PLATFORM-UI

## PROJECT OVERVIEW

This project is a front-end E-Learning Platform UI developed using HTML, CSS, and JavaScript. It includes course listings, embedded video lessons, and progress tracking functionality.

## FEATURES

- Multi-Page Interface
- Course Listing
- Video Embedding
- Progress Tracking
- Responsive Design
- Interactive UI

## TECHNOLOGIES USED

- HTML5
- CSS3
- JavaScript

## HOW TO RUN

1. Download or clone the repository.
2. Open index.html in a browser.
3. Navigate through the pages.
4. Watch videos and track course progress.

## CONCLUSION

The project demonstrates a modern E-Learning Platform interface with course management, video content integration, and interactive progress tracking.
